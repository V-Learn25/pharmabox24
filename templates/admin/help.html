Admin Help Guide

Getting Started

Pharmabox24 is your analytics portal for tracking prescription collection machine usage across all your pharmacies. As an admin, you can upload weekly data files, manage pharmacies and users, and view performance across the network.

The Dashboard is your home screen. It shows a summary of all pharmacy performance (today, 7 days, 30 days) and your most recent data uploads.

Uploading Data

  1. Click Upload Data in the sidebar.
  2. Choose your weekly data file (Excel .xlsx or .csv).
  3. Click Upload. The system will automatically read the file and import the records.
  4. You'll see a confirmation message showing how many records were imported.

Good to know:

  • The file should contain columns for S/N (serial number), Pharmacy Name, Date, Loaded Parcels, Collected Parcels, Removed Parcels, and Reminders Sum.
  • If the file contains an hourly distribution section ("Collected Parcel Distribution"), that data is imported too.
  • If a pharmacy serial number in the file doesn't exist yet, it will be created automatically.
  • Uploading data for a date that already exists will update (overwrite) the previous values for that date.
  • If a pharmacy has a notification email set, they'll receive an email when new data is uploaded for them.

Managing Pharmacies

Click Pharmacies in the sidebar to see all registered pharmacies.

  • Add a pharmacy: Click "Add Pharmacy" and enter the serial number (must match the S/N in your data files), pharmacy name, and optionally a notification email.
  • Edit a pharmacy: Click the edit icon next to a pharmacy to update its name or notification email.
  • View analytics: Click the eye icon to see that pharmacy's detailed statistics and recent data.
  • Delete a pharmacy: Removes the pharmacy and all its data. This cannot be undone.

Note: Most pharmacies are created automatically when you upload data. You only need to add them manually if you want to set them up before any data is uploaded.

Managing Users

Click Users in the sidebar to manage who can log in.

There are two types of user:

Admin

Can upload data, manage pharmacies and users, and view all analytics across every pharmacy.

Pharmacy User

Can only see the dashboard for the pharmacy they are linked to. No access to admin functions.

To create a pharmacy user:

  1. Click Add User.
  2. Enter their email address and name.
  3. Set a password (minimum 6 characters). Give this to them so they can log in.
  4. Set the role to Pharmacy User.
  5. Select their Pharmacy from the dropdown.
  6. Click Save.

To edit a user, click the edit icon. Leave the password field blank to keep their existing password.

Viewing Analytics

From the Dashboard, click the eye icon next to any pharmacy to view their detailed analytics, including:

  • Summary cards -- today, yesterday, last 7 days, and last 30 days totals
  • Daily trends -- a chart showing loaded, collected, and removed parcels over the last 30 days
  • Day of week patterns -- average parcels by day of the week
  • Hourly distribution -- when collections happen (morning, afternoon, evening, overnight)

Pharmacy users see this same dashboard when they log in, but only for their own pharmacy.

Passwords

  • Change your own password: Click your name in the top-right corner, then "Change Password".
  • Reset a user's password: Go to Users, click edit on their account, and enter a new password.
  • Forgot password: On the login page there's a "Forgot Password" link. This sends a reset email (if email has been configured).

Tips

  • Upload data regularly (weekly) to keep the analytics up to date.
  • The upload history is shown on the Upload Data page so you can see what's already been imported.
  • You can re-upload a file safely -- it will update existing records rather than creating duplicates.
  • Set a notification email on each pharmacy if you want them to be alerted automatically when new data is available.
